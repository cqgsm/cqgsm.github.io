git服务器代码自动部署

版本控制
  • git

每次本地代码都要手动上传到服务器在部署，对于懒人的我来说简直不能忍，于是折腾了下git自动化部署，期间踩了不少坑，记录一下防止下次遇到。

  • 服务器版本： ubantu 16.04
  • ipaddress：请填上自己服务器地址
  • 每次操作都要输入密码，需要配置密钥认证，后期再来填坑
  • 由于是个人项目，所以没有加入权限控制，后期填坑
